# 天体系统模拟器 - 项目概述

## 项目简介
这是一个基于Web的3D天体系统模拟器，让用户能够创建、学习和探索宇宙天体运行规律。应用采用现代Web技术，结合Three.js进行3D渲染，提供直观的天体物理学习体验。

## 核心功能特性

### 1. 3D宇宙模拟器
- 实时3D渲染的天体系统
- 支持多种天体类型：恒星、行星、卫星、小行星等
- 基于牛顿引力定律的物理引擎
- 可调节的时间流速和视角控制

### 2. 天体创建与管理
- 用户可自定义添加天体
- 设置天体的质量、速度、轨道参数
- 实时修改天体属性并观察变化
- 支持预设天体模板

### 3. 教育与学习功能
- 天体物理知识展示
- 引力作用可视化
- 轨道力学演示
- 实时物理数据展示

### 4. 预设场景库
- 太阳系模拟
- 双星系统
- 星系碰撞
- 自定义场景保存与加载

## 技术架构

### 前端技术栈
- HTML5 + CSS3 + JavaScript
- Three.js - 3D图形渲染
- Anime.js - 动画效果
- ECharts.js - 数据可视化
- 响应式设计

### 核心算法
- 牛顿万有引力定律计算
- 数值积分方法（Euler/RK4）
- 碰撞检测与处理
- 轨道预测算法

### 用户界面
- 现代化科幻风格设计
- 直观的控制面板
- 实时数据显示
- 交互式3D场景

## 页面结构

### 主页面 (index.html)
- 3D模拟器主界面
- 实时控制面板
- 天体管理界面

### 学习页面 (learn.html)
- 天体物理理论知识
- 交互式教学内容
- 公式和原理展示

### 场景页面 (scenarios.html)
- 预设场景库
- 场景详情和预览
- 自定义场景创建

## 目标用户
- 天文爱好者
- 物理学生
- 教育工作者
- 科普学习者

## 教育价值
- 直观理解天体运动规律
- 探索引力相互作用
- 学习轨道力学原理
- 培养科学思维能力

这个项目将结合视觉震撼的3D效果和严谨的科学计算，为用户提供一个既有趣又富有教育意义的天体物理学习平台。
